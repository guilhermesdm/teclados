# Tchaikeebs

Teclado low profile 6x4+3 (54 keys), totalmente impresso em PLA 3D e handwired com a logo do SPFC em baixo.

# Materiais

2KG - Filamento PLA

12 - Insertos M3x6
12 - Parafusos M3x6
02 - RP2040 Zero
01 - Cabo TRRS
02 - Sockets TRRS
54 - Keycaps low profile (usei KLP Lame Keycaps impressas em PLA - https://github.com/braindefender/KLP-Lame-Keycaps)
54 - Low profile switchs redragon
54 - 1N4148 Diodos
10M - 16 AWG fio de cobre (sem esmaltacao eh melhor)
10M - 28 AWG fio de cobre heat resistant
01M - Heat shrink tubes
01 - USB C transferencia de dados

Ferro de solda, melhor com temperatura ajustavel para usar nos insertos
Estanho 0.8mm
Plier
Tweezer
Terceiro braco de solda (opcional pois facilita)
Demais necessidades de solda
